> A Full-Stack, AI-Powered Fitness and Nutrition Platform (Web & Native Android)

**Inside** is a comprehensive fitness application designed to act as a personalized, AI-driven personal trainer. Users can onboard with their specific physical metrics and goals, and the application dynamically generates customized workout splits and calculated diet plans. 

To ensure maximum accessibility, the platform is fully responsive on the web and compiled as a native Android application using Capacitor.

## ‚ú® Features
* **Dynamic Generation:** Algorithms calculate caloric needs and generate specific Gym or Home workout routines based on user preferences.
* **AI Coach Integration:** A built-in, conversational AI assistant powered by the Google Gemini API to answer fitness, form, and nutrition questions in real-time.
* **Progress Tracking:** An intuitive daily logging system for weight, steps, diet costs, and workout completion.
* **Smart Meal Tracking:** Add meals and automatically estimate macro-nutrients using built-in estimation logic.
* **Cross-Platform:** Built as a React web application and successfully bridged into an installable native Android `.apk`.
